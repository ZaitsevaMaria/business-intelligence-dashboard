# business-intelligence-dashboard
В рамках учебного проекта были сгенерированы данные по продажам магазина электроники. Данные моделируют информацию, получаемую из нескольких таблиц базы данных с использованием операций объединения (join).

По результатам анализа были сделаны следующие выводы:
1.	Бизнес демонстрирует устойчивую и диверсифицированную структуру выручки: выручка распределена достаточно равномерно между каналами (Partner, Online, Retail) и регионами (North, South, West, East). Это свидетельствует об отсутствии зависимости от одного канала или рынка и, как следствие, о низком операционном риске. Такая структура может быть результатом как осознанной стратегии диверсификации, так и отсутствия фокуса на наиболее прибыльных сегментах. В последнем случае рекомендуется дополнительно проанализировать эффективность маркетинговых кампаний.
2.	Динамика выручки по регионам неоднородна: периоды максимальных и минимальных значений приходятся на разные месяцы в зависимости от региона. Это может свидетельствовать о влиянии локальных факторов (спрос, маркетинг, логистика), которые требуют отдельного анализа на региональном уровне.
3.	Лидером по объему продаж является категория Tablet, в то время как категория Laptop демонстрирует наименьшие показатели. Это может указывать на различия в спросе, ценовой политике или конкурентной среде для отдельных категорий товаров.
4.	При анализе выручки по товарным категориям в разрезе месяцев наблюдаются выраженные периоды роста и снижения.
Такая волатильность может быть связана с:
•	сезонным спросом, 
•	маркетинговой активностью, 
•	перебоями в поставках или логистике. 
Для повышения стабильности выручки рекомендуется:
•	усиливать маркетинговую активность в периоды спада,
•	оптимизировать распределение рекламного бюджета, 
•	контролировать процессы поставок и наличие товаров.
